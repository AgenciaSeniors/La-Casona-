# La Casona - Menu Digital

Menu digital para el restaurante **La Casona** con panel de administracion.

**Sitio:** https://agenciaseniors.github.io/La-Casona-/

## Caracteristicas

- Menu visual con categorias y productos
- Panel de administracion con Firebase
- CRUD completo de productos
- Diseno responsivo movil/desktop
- GitHub Pages hosting

## Tecnologias

- HTML5 + CSS3
- JavaScript
- Firebase Auth + Firestore
- GitHub Pages

## Uso

1. Abre el menu: https://agenciaseniors.github.io/La-Casona-/
2. Admin: /login.html
